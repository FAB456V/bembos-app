# Escáner QR externo

1. Ejecuta `npm start`.
2. Abre `http://localhost:4173`.
3. Copia `IOT_DEVICE_API_KEY` desde `wokwi-esp32-pickup/secrets.h`.
4. Elige cámara o archivo y escanea el QR.
5. Wokwi lo recibirá automáticamente en menos de dos segundos.

La cámara del navegador solo funciona en `localhost` o HTTPS.
